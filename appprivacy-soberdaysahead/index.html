This one's going to be pretty simple.  Sober Days Ahead does not ask for ANY enhanced permissions, nor does it ask to allow App Tracking.   I don't do any of that stuff.    None.   The App does not feature any advertising SDK code that could also be used to track you, or build some kind of marketing profile about you.    

This app was written to help ME, and hopefully help YOU.   It was not written to make any money.  If you feel inclined, give it some stars and/or a review on the App Store, but it should NEVER ask or prompt you to do that.  

The App Store itself will give me metrics like "300 people who saw your app on the store were in the UK" but it doesn't tell me anyhing that even remotely identifies you.

The date picker is a bit cranky at times, so it helps to pick the year first, then month, then day.
